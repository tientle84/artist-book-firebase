# Artist Book - React Native Project

<p>  
  <!-- Android -->

It includes:

-   Expo SDK `40.x.x`
-   Navigation using `react-navigation` 5.x.x
-   Firebase auth as backend for email auth
-   Firestore as backend for database
-   React Hooks
